# raytracer

A simple realistic renderer based on monte carlo path tracing.

####Requirement:
Qt4 with MinGW for Windows

####Features:
Multi-threads to speed up

Soft shadows

Depth of filed

Texture on planes

KD-Tree for faster intersection with obj models
